Installation
------------

### Windows

Copy the folder `ProjectLauncher` into `C:\Users\<username>\.qgis2\python\plugins`.
> The files `Makefile` and `resources.qrc` are not necessary.

### Linux
Build the plugin with Makefile:

```bash
make deploy
```
